# WARUNG

## Instalation
- clone repo 
```
git clone https://github.com/idkraf/app-transaksi-laravel.git
```

- install composer
```
composer install  --ignore-platform-reqs
```

- install node.js
```
npm install
```

- build assets or compile
```
npm run build
```

- configure the database on .env
- migration
```
php artisan migrate:fresh --seed
```
